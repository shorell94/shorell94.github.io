Posted on 2019-02-21

　　在《最好的告别》一书的总序中，我认识到了达芬奇手术机器人，不禁感叹现代技术的发展之快。人类，仿佛无所不能。但读的越深入，我越发的了解到，实际上，在人生这条单行道上，衰老与死亡是最终的归宿。强大如达芬奇机器人，也只能解决特定的故障。强行干预，更大概率上是在剥夺正常的安静宁和。

　　无独有偶，我最近也越来越相信有些事就是命，强求不来。尝试了，失败了，便不必一味懊责。接受生命的每种可能性，放过纠缠不清的自己。

　　一周前，我为内心的死结求助心理医生。那时，我固执的认为，自己长期以来逃避问题的方式，导致问题本身没有得到解决；因而，日积月累之后，内心终于承受不住困惑与痛苦；进而对自己的精神生活造成强烈的冲击。我无声的痛哭，却在心理医生一句“有些问题就是解决不了的呀，接受就可以了”后，戛然而止。我愣住了，一扇新的大门缓缓向我打开。原来，还是自己偏执了，谁的心中都有这样那样的伤痕，但随着时间的推移，世界越来越大，伤痕所占的相对空间越来越小。当你从年少时一碗水慢慢发展成一片汪洋，任那小石子棱角分明，却也难以对你造成实质上的伤害。人，不会永远活在年少。

Posted on 2019-02-19

　　有的人生像涓涓细流，轻声慢语地流淌；有的人生似是滔滔江河，由四面八方的细流汇聚。前者如归园田居，平静淡泊；后者如骄傲水龙，一泻千里。

　　诸葛亮，隐居茅庐多年；但在得到刘备的诚意之后，以傲然之姿出山，终得美名无数，流传千古。试想乱世之中，诸葛若执意隐世，他又如何为自己的人生书写那浓墨重彩的一笔？

　　年轻人，同样应当选择后者。

　　数年前，父母为你的灵魂套上躯壳，你得以降临人世。几十年后，你的躯壳化为一抔黄土，泼洒于天地之间。这副皮囊还于大地，唯有这几十年的磨难与成长融于灵魂。

　　如何敢白白走这一遭？唯有抓住这个让你得以经历的机缘，做一条有宽度、有厚度的大河是尔。

Q：
人在什么状态下会拒绝进步？
A：
1.盲目乐观，坚信车到山前必有路，明天一定会更好。
2.有限的努力难以成功，顽固的懒惰却如影随形。
3.在一段时间的努力后，却发现只是无用功，于是在绝望之下放弃挣扎。

　　只有自己为之痛苦过的经历，才会永远刻在脑海里。而一点点敲下这三种状态下的我，正是它们的结合体。

　　从前天晚上抵家到现在，已经两天。在这个可以让我无所顾忌的环境中，我的臭毛病隐隐的回归了。

　　现在的我一个丧字再贴合不过了。时时刻刻觉得累，分分秒秒感受躁。

　　这真的不是个好征兆。

　　劝劝自己。

20190118

Posted on 2019-01-18

　　宋运萍下线了。一个挺着大肚子的孕妇，盯着钢筋卸货时，不小心被绊倒了，大出血，不治而亡。有人说她是自己作，说不会保护自己的人是最无能的。我虽痛宋运萍逞强，但她顾全大局的隐忍、和她为了好日子奋斗的劲儿，却每每让我感怀，指责的话总是难以说出口。

　　她的身上，其实有一股我妈妈的劲儿。

　　妈妈跟宋运萍一样，是个读书人。也是因着读书误了年龄，便找个差不多的就嫁了。我们村没几个上了高中的，几个大爷家的孩子，好一点的上完了初中，大多数却是读了小学便辍学了。而我的妈妈在我正式上学前，就教会了我英文字母歌。妈妈叹息众人的文化水平，说老家教育不行，我们宝宝以后要出去读书。现在想想，可能我也是后来家里一步步奋斗的原因之一吧。

　　后来，家里做起了教辅书的生意。我晕车的妈妈，每周都要坐几次一个多小时的班车去县里。也常有深夜，我和妈妈还在一楼那个堆满各种各样教辅书的房间里，清点归类。我至今还清楚地记得我跟妈妈在一楼堆满书的房间里清点到深夜的情景。那时我上小学，动作麻利，常在一捆捆书上爬来爬去，有时还需要发动我骄傲的小臂膀挪动一番。那时啊，我的妈妈说宝宝真能干。

　　想想宋运萍一家和我们多么相像，都是奔着越过越好去的，都讲道理，都人美心善。他们在八几年遇到的苦楚，我们同样经历过，而我的妈妈，也是从那个阶段挺过来的。

　　或许好人总是多磨，事实一步步告诉你生活不易。我啊还没长大，弟弟更是幼稚，而我那五十多岁的妈妈，还在独自撑着整个家。

　　有人为了你一直在努力，你凭什么懈怠。

人物设置：
　　新人：1.有轻微社交恐惧症，不知如何跟办公室的人相处，故而假装淡定。
　　　　　2.性格使然，有些懒惰，没人给安排工作，便乐得清闲。

　　老人：1.强权政治，性格暴躁。

　　　　　3.德高望重，技术中流但资历老，圆滑老道。

　　“你今天先不忙着正式接手工作，先熟悉熟悉工作环境，有什么不懂的就问老于。”陈主任…

中途废止：
　　我自认在人物设置上不差，情景设想上也逊色不了多少，怎奈何一味想周全每个细节，反而难以突出重点了。无独有偶，平日做事同样存在这个毛病，故而事事慢，进而事事拖。“老太太的裹脚布，又臭又长。”如是而已。

初步认识命令行

1. 命令历史窗口：“F2”可以显示更多的历史信息。
2. 新建文件：命令行中键入“NEW”后按Enter键，弹出“选择样板”对话框，选择后即可新建文件。
    ***** 样板的选择 *****
    2.1 二维
        ①“acadISO-Named Plot Styles”使用“命名打印样式”
        ②“acadiso”使用“颜色相关打印样式”
    2.2 三维
        ①“acad-ISO-Named Plot Styles3D””
        ②“acadiso3D”    
     ***** the end *****    
3. 保存文件：命令行中键入”Save“后按Enter键，即可保存文件。
4. 清理文件：命令行中输入”Purge“后按Enter键，即可清理垃圾文件。
...

常用命令

1. 设置点样式：Ddptype
2. 绘制单个点：point
3. 定数等分点：divide
4. 定距等分点：measure
5. 设置图形界限：limits
    ***** steps（以二维为例） *****
    ①limits
    ②0,0 Enter键（设置左下角为0,0）
    ③500,500 Enter键（设置右上角为500，500）
    ***** the end *****
6. 绘制直线：line
    ***** steps（以二维为例） *****
    ①line
    ②0,0 Enter键（设置第1个点为0,0）
    ③320,320 Enter键（设置第2个点为320，0）
    ④@320<120 Enter键（设置第3个点与第2个点所连线段：长度为320，与平行线夹角为120）
    ⑤c Enter键（封闭图形）
    ***** the end *****
7. 绘制多段线：Pline
    ①角度以整张图为参考对象（“<300”=“<-60”）
    ②坐标是以上一个点为参考对象（上一个点为原点）
8. 编辑多段线：Pedit
9. 绘制多线：Mline（由“两条或两条以上的平行线元素”构成）
